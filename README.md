# Рукогон - руководитель гонки
## Игра полностью написана ИИ, человек выступал лишь оператором.

Статическая аркадная игра на React с видом сверху.

Игрок управляет руководителем гонки на дрифт-соревновании. Его задача — перемещаться между возникающими проблемами, успевать их решать и не допускать перегруза площадки. Если проблем становится слишком много или одна из них "сгорает", игра заканчивается.

## Что есть в проекте

- Полностью клиентское приложение на `React + Vite`
- Сохранение локального состояния в `localStorage`
- Статическая карта с редактором тайлов
- Импорт и экспорт пользовательской карты в JSON
- Автоматический режим наблюдения
- Подвижные проблемы-боты
- Пиксельарт-иконки для разных типов проблем

Серверная часть не требуется: серверу достаточно отдавать статические файлы из каталога сборки.

## Игровая логика

На карте есть 5 зон:

- Трасса
- Штаб
- Трибуны
- Касса
- Паддок

Типы проблем:

- Происшествия на трассе
- Документы в штабе
- Зрители на трибунах
- Злоумышленники на кассе
- Споры в паддоке

Особенности:

- Перемещение возможно только по дорогам и тропинкам
- В режиме редактора карту можно перекрашивать по тайлам
- Подписи зон можно двигать
- Карту можно сохранить в файл и загрузить обратно

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска Vite покажет локальный адрес в терминале. Обычно это что-то вроде:

- `http://localhost:4173`

## Сборка production

```bash
npm run build
```

Готовая статика будет собрана в каталог `dist/`.

## Локальный просмотр production-сборки

```bash
npm run preview
```

## Публикация на статический хостинг

Проект подходит для выкладки в любой статический хостинг:

## Управление

В игре:

- `W`, `A`, `S`, `D`
- стрелки

В интерфейсе:

- `Старт` — начать новую игру
- `Пауза` — остановить текущую игру
- `Режим редактирования` — перейти к редактированию карты
- `Звук` — переключить звук
- `Автобот` — включить автоматическое управление
- `Подвижные проблемы` — включить/выключить мобильные проблемы

## Редактор карты

В редакторе можно:

- выбирать тип тайла
- рисовать мышью по карте
- двигать подписи зон
- сохранять карту в файл
- загружать карту из файла

### Формат файла карты

Редактор сохраняет JSON со структурой:

```json
{
  "version": 1,
  "mapRows": ["..."],
  "labelPositions": {
    "headquarters": { "x": 120, "y": 130 },
    "stands": { "x": 810, "y": 92 },
    "ticketOffice": { "x": 118, "y": 418 },
    "paddock": { "x": 760, "y": 455 },
    "track": { "x": 448, "y": 58 }
  }
}
```
